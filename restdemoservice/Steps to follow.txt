1. create a dynamic maven project with archtype: maven-webapp.
2. Configure web.xml to specify the <context-param>,<listener>,<servlet>,<init-param> and <servlet-mapping> .
these are mandatory tags to be set.
3. set the springConfig.xml file for bean initialization.
4. set the pom.xml file. Specify the following:
 artifactid,groupid,packaging , maven plugin for compilation, tomcat7-maven-plugin for deployment , specify the spring jersey dependencies.

5. implment business logic.

6. Run on tomcat server.

7. Use latest soap UI  or Advanced Rest client app for chrome to hit the service.
